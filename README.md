# Coffee app. 

## Overview
A simple mobile app that allows users to log and track their daily coffee consumption with quick tap counters and basic analytics.

## Problem
People lose track of how much coffee they consume throughout the day and lack visibility into their consumption patterns over time

## Target User
Coffee drinkers who want to monitor their daily caffeine intake for health or habit awareness

## Features
- Quick cup counter with +/- buttons
- Daily coffee log with timestamps
- Weekly/monthly consumption trends
- Simple statistics dashboard
- Basic cup size options (small, medium, large)
